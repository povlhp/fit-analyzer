=FIT file Analyzer

This project analyzes .fit files, and shows data/graphs.
You can upload multiple files, .fit files or .zip files where the first .fit file is used. .zip file support was added since Garmin Connect deliver .zip files.

All this is vibe coded, in OpenCode CLI, so in case the code looks bad, you know why.
This is my first attempt at vibe coding something more complex, and since I don't do much Node/HTML on a daily basis,
it was way faster than doing it by hand.  The main *disadvantage* is, that I did not learn much. 

If I had developed the classic way, I would have spent at least 2-3x the time, but I would heave learned about using chart.js, the Garmin FIT parser library etc, and might have found more data in the files I could present.

So I got a product that, with lots of iterations, has progressed, and backstepped, to something that is quite usable,
but of a quality where I am not sure if I can maintain the code by hand.
